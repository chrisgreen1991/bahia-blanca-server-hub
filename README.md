## What it does

Bahia Blanca is a web-based administration dashboard made for managing FiveM roleplay servers. It is meant to provide administrators with a browser workspace for routine server supervision, control actions, and fast access to current operational information, without needing a separate desktop application.

The project is implemented in HTML and follows a responsive layout that works across desktop and mobile screens. It includes live server monitoring, resource management, player moderation views, console access, and configuration editing, so it can cover the core tasks involved in day-to-day server administration.

## Features

- Live server status monitoring for ongoing operational visibility
- Resource controls for start, stop, and restart commands
- Player list access with moderation-focused tools
- Live console view for reviewing server output
- Server configuration editor for direct browser-based changes
- Responsive interface built for desktop and mobile browsers
- Lightweight HTML implementation for simple deployment
- Secure administrator session handling for controlled access
